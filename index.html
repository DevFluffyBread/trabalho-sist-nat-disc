Como usar

Variáveis: Use A–Z para representar proposições.

Operadores:

  • ^ (AND): verdadeiro quando ambos são verdadeiros
  • v (OR): verdadeiro quando pelo menos um é verdadeiro
  • ∼ (NOT): inverte o valor lógico
  • → (IMPLICA): falso só quando P é V e Q é F
  • ↔ (BICOND.): verdadeiro quando os dois têm o mesmo valor
